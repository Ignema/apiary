# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-11-26

### Added
- Initial release of Apiary MDE toolkit
- Custom DSL for REST API specification
- ANTLR4-based parser for DSL
- AST to Ecore metamodel transformation
- OCL-inspired semantic validation
- Jinja2-based code generation for FastAPI applications
- SQLModel ORM support with SQLite and PostgreSQL
- CLI interface for code generation
- Comprehensive test suite (unit and integration tests)
- Example DSL files (PetStore, TaskManager)
- Complete documentation and usage examples

### Features
- Parse DSL files into Abstract Syntax Trees
- Transform AST to validated metamodel instances
- Generate production-ready FastAPI applications
- Automatic CRUD endpoint generation
- Database schema generation with SQLModel
- UV-based dependency management for generated projects
- OpenAPI documentation (Swagger UI and ReDoc)
